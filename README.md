# Modelo de site para salão de beleza
Este é um modelo de uma pagina web desenvolvida para um salão de beleza, utilizando as linguagens HTML CSS e JavaScript. Utilizando conceitos de UX-UI paramelhor comunicação e interação com o usuario.

A pagina possui apenas um arquivo HTML onde dispoe de todo conteudo apresentado, incluindo botoes e cards com descrição detalhada dos serviços prestados.
A navegação é dinamica e a aparencia é agradavel de olhar graças ao CSS e o JavaScript que trazem uma aparencia mais moderna para a pagina.
